HARI MWOL BIRTHDAY WEBPAGE
==========================

WHAT THIS IS
------------
A fully local birthday webpage made with:
- index.html
- style.css
- script.js
- assets/ (all photos used by the page)

HOW TO OPEN IT
--------------
1. Extract the ZIP file.
2. Open the folder.
3. Double-click index.html
   OR right-click index.html -> Open with Chrome / Edge / Safari.

It works locally. No special software is needed.

HOW TO EDIT TEXT
----------------
Open index.html in any text editor.
You can edit:
- title
- headings
- paragraphs
- button text
- birthday note

HOW TO CHANGE PHOTOS
--------------------
1. Put your replacement image inside the assets folder.
2. Keep the same file name if you do not want to edit the HTML,
   OR change the file name inside index.html.

OPTIONAL FREE HOSTING (ONLY IF YOU WANT)
----------------------------------------
If later you want to put this online for free:

Option 1: Netlify
- Go to https://app.netlify.com/drop
- Drag the extracted folder there
- It will give a free live link

Option 2: GitHub Pages
- Create a GitHub repo
- Upload all files
- Enable GitHub Pages in repo settings

IMPORTANT
---------
If you move index.html out of the folder, keep the assets folder with it.
Otherwise the photos will not load.
